# Python-Class
Here is a repository containing all the classwork I completed during my Intro to Logic and Programming course at NSCC. In this class we learned the basics of programming, so this coursework will be simple and basic. We completed this course using Python as our programming language. The instructions for each assignment is in each file as comments. 

* Exercises were short assignments that weren't worth as much of our grade, and were usually quicker to complete.
* Tech-Checks were what our tests were called. We had one hour to complete these in class.
* Assignments were made up of multiple parts, done at the end of each unit.

The 'Files' folder, contains files necessary for Assignment 4 which includes writing and reading files.

Thank you for viewing this. I had a lot of fun in this class, and each assignment felt like a fun puzzle rather then work.
